Telemetry IoT-eH Health Monitoring and Wireless Sensing Systems:

 


The Objectives of the project involves:


  • Portable ultrasound system with customized features for Rural healthcare system.

  • Internet of Things for Physiological Monitoring Systems with screening diagnostics for cardiovascular, kidney, liver etc.

  • Cloud based distributed medical data repository.

  • Enabling technologies for Body and Presronal Area network.
